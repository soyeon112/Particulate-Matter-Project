### 📁 프로젝트 소개

이 프로젝트는 공공데이터 API를 활용한 프로젝트 입니다. <br/>
지역별로 측정소 데이터를 활용하여 미세먼지 현황을 확인할 수 있으며 <br/>
관심있는 지역의 측정소를 즐겨찾기 하여 한눈에 모아볼 수 있습니다. <br/>
